Problem Set 0
======================

Elapsed Time Calculator
----------------

定义函数elapsed(seconds), 打印秒数所对应的时分秒。

例如1000秒对应0小时，16分钟，40秒。如果调用elapsed(1000)，屏幕上会打印三行数据：
0
16
40
