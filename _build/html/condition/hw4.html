2. Function with if statement

Write the following functions to your editor. Describe what each of the functions does, which you will add it besides the function as comment using #.

def getGPA(grade):
    if(grade > 90):
        return 'A'
    if(grade > 80):
        return 'B'
    if(grade > 70):
        return 'C'
    return 'D'

def magic_string(s):
    s1 = s[0:len(s)//2]
    s2 = s[len(s)//2:]
    return s1 == s2

Narcissistic Number

Narcissistic Numbers are defined as follows: An n-digit number is narcissistic if the sum of its digits to the nth power equal the original number. For example with 3 digits, say I choose the number 153: \(153 = 1^{3} + 5^{3} + 3^{3}\). So 153 is a Narcissistic Number.

Write a function called check(), to determine if a 3-digit number which user input is a Narcissistic Number. For example:

  • input: 153

  • output: “153 is a Narcissistic Number”

  • input: 165

  • output: “165 is not a Narcissistic Number”
